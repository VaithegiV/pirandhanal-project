# Happy Birthday Eyalarasan 🌿

A romantic birthday website.

## Setup
1. Add your 3 photos inside the `images/` folder:
   - `images/couple.jpg`   → the selfie photo (used as hero)
   - `images/baby.jpg`     → the baby photo
   - `images/travel.jpg`   → the sandals/travel photo
   - `images/together.jpg` → the hand-holding photo

2. Open `index.html` in a browser to preview.

## Push to GitHub Pages
```bash
cd bday-swe
git init
git add .
git commit -m "birthday website"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/bday-swe.git
git push -u origin main
```
Then go to repo **Settings → Pages → Source: main branch** and it will be live!
